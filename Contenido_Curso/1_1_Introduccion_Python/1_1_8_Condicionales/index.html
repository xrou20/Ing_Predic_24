    Condicionales

    if - else

    La forma general de escribir es: if-else, en código if-else, siendo else opcional.

    if condicional:
    <bloque de sentencias, ejecutado cuando la condición if es Verdadera>
else:
    <bloque de sentencias, ejecutado cuando la condición if es Falsa>

    También es posible una composición con múltiples if-else:

    if condicional1:
    <bloque de declaración>
elif condicional2:
    <bloque de declaración>
elif condicional3:
    <bloque de declaración>
else:
    <bloque de declaración>
<próxima declaración>

    Ejemplo con múltiples if-else

    Una función definida por partes:

    N ( x ) = { 0 , x < 0 x , 0 ≤ x < 1 2 − x , 1 ≤ x < 2 0 , x ≥ 2

    La implementación de Python con múltiples if-else :

    x = float(input("Ingrese el valor de x: "))

if x < 0:
    print(0)
elif 0 <= x < 1:
    print(x)
elif 1 <= x < 2:
    print(2 - x)
elif x >= 2:
    print(0)
